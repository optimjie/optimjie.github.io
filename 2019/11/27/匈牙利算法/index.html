匈牙利算法

发表于 2019-11-27 更新于 2019-11-29 分类于 AcWing基础课 阅读次数： Valine：
本文字数： 1.3k 阅读时长 ≈ 1 分钟

匈牙利算法是一种在多项式时间内求解任务分配问题的组合优化算法，并推动了后来的原始对偶方法。美国数学家哈罗德·库恩于1955年提出该算法。此算法之所以被称作匈牙利算法，是因为算法很大一部分是基于以前匈牙利数学家Dénes Kőnig和Jenő Egerváry的工作之上创建起来的。
